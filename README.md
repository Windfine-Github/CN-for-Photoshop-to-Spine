# CN-for-Photoshop-to-Spine
汉化了EsotericSoftware的Photoshop to Spine插件

## 安装  
  1.打开你的PS安装目录'Adobe Photoshop\Presets\Scripts'，将PhotoshopToSpine.jsx放入即可  
  2.完成安装
